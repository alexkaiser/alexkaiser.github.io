Github profile.



Open source projects on which I am the primary author:

heart_valves:

Model generation and FSI for aortic and mitral valves. Includes code that calls IBAMR for fluid structure interaction, scripts, plotting, movie generation.

mc_stretch:

Fast, affine-invariant, GPU parallel MCMC

SimplifySum: automatic simplification of symbolic sums in Mathematica


Open source projects on which I contribute:


SimVascular:  comprehensive tools for modeling and simulation of patient-specific vascular flows


IBAMR: immersed boundary adaptive mesh refinement, fluid-structure code interaction suitable for simulating heart valve mechanics


ARPREC: Arbitrary precision arithmetic in C++ with overloaded operators. Previously run by Lawrence Berkeley National Laboratory, now mirrored by David H. Bailey.


QD: Double-double and quad double precision arithmetic in C++ with overloaded operators. Previously run by Lawrence Berkeley National Laboratory, now mirrored by David H. Bailey.


MatrixFFT: Apple’s FFT library. Both library development and the advanced computation group who developed it appear to be discontinued since I worked with them in 2010.
